データサイエンスで切り拓く、共創の未来

私たちは、個人の可能性とQOLの最大化というパーパスを掲げています。この壮大な目標は、一社単独では達成できません。多様なドメイン知識、革新的な技術、そして社会変革への強い意志を持つパートナーとの協業こそが、未来を創造する原動力になると信じています。

人間中心のAI

私たちの技術は、常に「人」を中心に設計されます。個人の尊厳と自己決定権を尊重し、テクノロジーが人々の幸福に貢献するための倫理的基盤を最優先します。

最先端の技術力

ライフログ解析、因果推論、強化学習といった先端AI技術を駆使し、複雑な課題に対して高精度な予測と最適な意思決定支援を提供します。

透明性と信頼性

データ利用の透明性を確保し、アルゴリズムの公平性・説明可能性を追求します。パートナーやユーザーとの信頼関係こそが、持続的な価値創造の基盤です。

パートナーシップの可能性

貴社の持つ強みと私たちのデータサイエンス技術を掛け合わせ、新たなイノベーションを生み出しましょう。私たちは様々な形のパートナーシップを歓迎します。

共同研究

学術機関や研究機関と共に、Society 5.0が直面する未解決課題（個別化医療、生涯学習パスの最適化、スマートシティにおけるQOL向上など）に挑みます。因果推論、強化学習、連合学習などの先端技術を用いた社会実装を目指す研究を推進します。

事業提携

各業界のドメイン知識と私たちのAI技術を融合させ、既存事業のDX推進や新たなパーソナライズサービスの共同開発を目指します。データ利活用による顧客体験の向上、業務効率化、新規収益源の創出を実現します。

技術・データ連携

高品質なデータアセット、革新的なアルゴリズム、高性能なインフラ技術をお持ちの企業と連携します。相互の技術を連携させることで、より強固なデータエコシステムを構築し、社会全体に大きなインパクトを与えるソリューションの創出を目指します。

未来を創るプロフェッショナルへ

私たちのミッションに共感し、データとAIの力で社会をより良くしたいという情熱を持つ人材を求めています。最先端の課題に挑戦し、共に成長できる環境がここにあります。

データサイエンティスト

Data Scientist

Web上の多様なライフログデータから価値ある洞察を引き出し、統計モデリングや機械学習モデル開発を通じて、個人の意思決定を支援するアルゴリズムを構築します。

主な責務: 課題設定、データ分析、予測モデル開発、因果推論による効果検証、ABテスト設計・分析

必須スキル: Python (Pandas, scikit-learn, etc.), SQL, 統計学・機械学習の深い知識

AI/MLエンジニア

AI/ML Engineer

深層学習を含む最新のAIモデルを研究開発し、大規模データ処理基盤上で安定的に運用可能なシステムとして実装します。MLOpsの観点からモデルのライフサイクル全体を管理します。

主な責務: AIモデルの研究開発、MLパイプラインの構築・運用、パフォーマンス最適化、新規技術の調査・導入

必須スキル: Python (PyTorch/TensorFlow), MLOpsプラクティス, クラウド環境 (AWS/GCP/Azure) でのモデル開発・運用経験

データエンジニア

Data Engineer

信頼性と拡張性の高いデータ基盤を設計・構築・運用します。分析やモデル開発に必要なデータを、安全かつ効率的に利用できる形に整備するデータパイプラインを構築します。

主な責務: データパイプライン (ETL/ELT) の設計・開発、データウェアハウス/レイクの管理、データ品質の担保

必須スキル: SQL, Python, 分散処理技術 (Sparkなど), クラウドデータサービスに関する知識
